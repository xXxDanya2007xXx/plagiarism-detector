# 📢📰 Summar-IT
**Summar-IT** - это инструмент для автоматического сбора и суммаризации (сокращения) 🗞️ новостей из <img src="https://upload.wikimedia.org/wikipedia/en/4/43/Feed-icon.svg" width="18"/> [RSS-лент](https://www.rssboard.org/rss-specification).  
✨ *Проект призван решить проблему информационного шума, помогая быстро узнавать ключевые события дня без чтения длинных статей.*

## 🌈 Функционал
- <img src="https://upload.wikimedia.org/wikipedia/en/4/43/Feed-icon.svg" width="18"/> **Парсинг RSS**: Сбор заголовков и ссылок из популярных IT-источников
- 🤓  **Умная суммаризация**: ...
- 🤖 **Автоматизация**: Скрипт запускается автоматически через [GitHub Actions](https://github.com/features/actions) по расписанию
- 📝 **Отчётность**: Генерация ежедневного дайджеста в формате `Markdown`

## 🚀 Установка и запуск
###  <img src="https://upload.wikimedia.org/wikipedia/commons/3/35/Tux.svg" width="24"/> Linux
```sh
# Клонируем репозиторий
git clone https://github.com/xXxDanya2007xXx/Summar-IT.git
cd Summar-IT

# Устанавливаем и активируем виртуальное окружение
python3 -m venv .venv
. .venv/bin/activate

# Устанавливаем зависимости
pip3 install -r requirements.txt

# Запускаем
...
```
> [!TIP]
> Для выхода из виртуального окружения используйте команду `deactivate`
